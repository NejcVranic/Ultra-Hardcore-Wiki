# Knitting Kit
Knitting Kit is a tool [item](../items.md) that can be used to make [Knitten Fabric](../items/knitten_fabric.md). 

<div style="display: flex; align-items: flex-start; gap: 24px;">  
<div style="flex: 1;">  
<p>A Kniting Kit starts with 0 durability and increases it every time it is used (right clicked). When the durability is filled, the item is replaced by Knitten Fabric.</p>
</div>
