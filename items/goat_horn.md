# Goat Horn
Goat Horn is a Vanilla [item](../items.md). The datapack adds a use case for the item, allowing it to change the weather.

If a player uses a goat horn while wearing a carved pumpkin on their head there is a roughly 33% chance the weather toggles between clear and thunder.

> This mechanic can be used in combination with a Trident enchanted with channeling, giving the player consistent access to lightning.
