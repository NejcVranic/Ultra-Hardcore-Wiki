# Glistening Honey
Glistening Honey is a tool [item](../items.md) that can be used multiple times to grand Regeneration to the player.

<div style="display: flex; align-items: flex-start; gap: 24px;">  
<div style="flex: 1;">  
<p>Every time the item is used it grants the player Regeneration 2 effect for 4 seconds.</p>
<p>The texture of the item changes as the durability decreases.</p>
</div>
